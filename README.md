# Проект 4: Место

### Обзор
Проект с фотокартачками ярких мест Российской Федерации с возможностью редактирования профиля.

Использовались следующие технологии
Flex
Grid-layout
Javascript

Планы по доработке:
- Возможность редактирования аватарки
- Добавление и удаление фотокарточек
